
# 简介
基于 CocosCreator 3.6.2 版本创建的 **3D寻路** 工程。

# 目前功能

    增加圆形障碍
    增加方形障碍
    增加角色
    选择终点
    增加连接线link
    清除所有障碍
    清除所有链接线link
    导出bin文件
    读取bin文件
    导出场景的obj文件

# 支持平台

    Web
    Android
    iOS
    Windows
    macOS
